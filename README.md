# py-tui-2048
py-tui-2048 is the popular 2048 game written with the curses library in python.

## Features
- Restart with r
- Quit with Ctrl-c
- Support for Arrow Keys, WASD, and HJKL (Vim Keybindings)
- Highscore saves
